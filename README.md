# 🎯 Loyalty Points Program dApp (Soroban + Stellar)

## 📌 Project Description

The **Loyalty Points Program dApp** is a decentralized application built using **Soroban Smart Contracts** on the **Stellar blockchain**. It enables businesses to reward users with loyalty points in a transparent, secure, and tamper-proof way.

This project was developed using **Stellar IDE**, combining a Rust-based smart contract with a simple decentralized application (dApp) interface for interaction.

## ⚙️ What it does

This dApp allows users and administrators to interact with a blockchain-powered loyalty system:

- Users can **earn loyalty points**
- Users can **redeem points for rewards**
- Anyone can **check points balance**
- All transactions are **stored on-chain**

The system removes reliance on centralized databases and ensures **trust, transparency, and immutability**.

## 🌐 dApp Usage

### 1. Connect Wallet
- Connect your Stellar wallet (e.g., Freighter)

### 2. Earn Points
- Call the `add_points` function  
- Enter user address and points amount  

### 3. Redeem Points
- Call the `redeem_points` function  
- Enter points to redeem  

### 4. Check Balance
- Call `get_points`  
- View current loyalty points  

---

## 📂 Smart Contract Functions

| Function         | Description |
|-----------------|------------|
| `add_points`    | Adds points to a user's account |
| `redeem_points` | Deducts points from a user |
| `get_points`    | Returns user's current balance |
